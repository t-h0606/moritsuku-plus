📡 直接掲載の設定（GitHubトークン）

「サイトに掲載」ボタンでarticles.jsonを直接更新するには、GitHubのアクセストークンが必要です。 GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token。 Repository accessで moritsuku-plus のみ を選び、Permissionsの Contents を Read and write にして作成し、 表示された「github_pat_」で始まる文字列を下に貼ってください。トークンはこの端末のブラウザ内にだけ保存されます。
